# Chrome Web Store 商店资料草稿

## 名称

TabTidy

## 概述

用右侧新建、右侧复制和可视化面板管理 Chrome 标签页。

## 详细说明

TabTidy 是一个帮助你整理 Chrome 标签页的小工具。

它可以把新建标签页放在当前标签页右侧，也可以把当前标签页复制到右侧，让相关页面自然待在一起，不再被丢到标签栏最末尾。

如果当前标签页位于 Chrome 原生标签组中，TabTidy 会把新建或复制出的标签页自动加入同一个标签组，保持你的工作上下文不被打散。

TabTidy 还提供一个全页面 Tab Board，用更宽敞的视图展示当前普通 Chrome 窗口中的标签页。你可以在面板中查看标签组、未分组标签页、本地缩略图和页面标题，并快速激活、移动、移出分组、关闭或拖拽标签页。

主要功能：

- 在当前标签页右侧新建标签页。
- 复制当前标签页到右侧。
- 保持 Chrome 原生标签组上下文。
- 通过 Tab Board 管理普通 Chrome 窗口中的标签页。
- 在面板中激活、移动、移出分组、关闭和拖拽标签页。
- 为最近可见页面显示本地缩略图。
- 对无法截图的页面显示字母占位符。
- 缩略图只保存在本机，不上传浏览数据。

隐私说明：

TabTidy 不收集、不上传、不出售、不共享你的浏览数据。标签页缩略图只保存在你本机的 Chrome 扩展存储中，不会发送到任何服务器。

## 隐私权做法填写参考

单一用途：

TabTidy 用于帮助用户整理 Chrome 标签页，包括在当前标签页右侧新建标签页、复制当前标签页到右侧，以及通过可视化 Tab Board 管理标签页。

权限说明：

- `tabs`：用于查询、创建、复制、移动、激活和关闭标签页。
- `tabGroups`：用于读取和维护 Chrome 原生标签组，把标签页加入分组或移出分组。
- `storage`：用于保存本地缩略图缓存。
- `unlimitedStorage`：用于避免大量本地缩略图缓存触发较小的存储限制。
- `activeTab`：用于在用户主动触发扩展后捕获当前可见标签页的本地缩略图。TabTidy 不读取网页正文内容，不修改网页内容，也不会上传数据。

数据使用：

TabTidy 不收集、不传输、不出售、不共享用户数据。Tab Board 使用的标签页标题、URL 和缩略图只保存在本机 Chrome 扩展存储中。

远程代码：

TabTidy 不使用远程代码。所有扩展代码都随扩展包一起发布。

## 审核测试说明

1. 安装扩展。
2. 打开一个普通网页。
3. 使用 `Ctrl+Shift+Y` 或 `Command+Shift+Y` 在当前标签页右侧新建标签页。
4. 使用 `Ctrl+Shift+L` 或 `Command+Shift+L` 复制当前标签页到右侧。
5. 创建一个 Chrome 原生标签组，重复上面两个操作，确认新标签页或复制页会加入同一分组。
6. 使用 `Ctrl+Shift+U` 或 `Command+Shift+U` 打开 Tab Board。
7. 在 Tab Board 中测试激活、移动、移出分组、关闭和拖拽标签页。
8. 确认最近可见的普通网页会显示缩略图，Chrome 内部页面会显示占位符。
